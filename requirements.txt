# yt-dlp floor: versions below 2026.7 miss current extractor fixes AND —
# the sharp edge — an OLD Python's pip resolves an unbounded "yt-dlp" to
# an ancient release (macOS system python3 is 3.9, for which "latest" is
# a 2025.10 build that can't use curl-cffi ≥0.10 and fails on modern
# YouTube). The floor makes such an install fail loudly instead of
# producing a silently-broken downloader. Upgrade freely above it.

# handler; 0.10–0.14 are the known-good range for current yt-dlp.
